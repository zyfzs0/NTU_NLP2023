2013年亞洲區彩妝界最大盛事，莫過於領銜全球的時尚精品集團LVMH路威酩軒集團正式收歸MAKE UP FOR EVER台灣經營權，品牌logo也於元月大方更新。受惠於代理權的終結，在萬物齊漲的浪潮之下，MAKE UP FOR EVER因應重整商品線的良機，76樣品牌商品價格平均調降近20%！其中32樣明星商品價格調降近10%，44樣專業彩妝師商品價格調降近30%，甚至有單樣商品最高降幅40%！更令人興奮的是，挾著居高不下的沸騰買氣，元月25日，MAKE UP FOR EVER將擴大服務，進駐新光三越南西店本館，屆時歡迎消費者現場體驗煥然一新的MAKE UP FOR EVER！回歸集團 LVMH引領MAKE UP FOR EVER展望亞洲
1984年法國藝術家Dany Sanz用心鑽研全方位彩妝商品以及揮灑彩妝藝術，成立MAKE UP FOR EVER品牌，自此法式彩妝就引領全球時尚圈，1999年時尚精品龍頭集團LVMH併購MAKE UP FOR EVER，台灣MAKE UP FOR EVER迷引頸期盼近13年，終於於2013年，MAKE UP FOR EVER台灣代理經營權也正式移交回全球時尚龍頭LVMH集團，是台灣消費者最貼近法式精神的一刻！

與巴黎零匯差 近80樣品牌商品 即日降價近20%
為了讓台灣消費者感受到MAKE UP FOR EVER台灣經營權被全球時尚巨人集團LVMH正式收回的實質優惠，即日起，MAKE UP FOR EVER台灣與巴黎同步零匯差！ 近80樣品牌商品平均調降近20%，包含底妝、唇彩、眼頰彩、特效商品、彩妝師專業工具等因為反應成本，進行價格調整，如：HD亮晰潤色隔離霜、高效防曬隔離霜SPF50/PA+++、HD無瑕粉底液、絲柔雙效粉餅、絲柔蜜粉餅、修容腮紅餅、HD漾彩腮紅霜、晶亮豐盈唇露等…，32樣明星商品價格調降將近10%，44樣專業彩妝師商品及工具價格調降近30%，甚至部分單品價格調降40%！高比例價差比起周年慶有過之無不及，讓消費者過足了癮。
嶄新2013！MAKE UP FOR EVER再躍 進駐新光三越南西店本館 
更讓人興奮的是，為擴大對台灣消費者的服務，MAKE UP FOR EVER於2013年元月25日即將於新光三越南西店本館盛大開櫃；未來，MAKE UP FOR EVER除了在通路上滿足消費者的便利性，嚮往專業彩妝的台灣朋友可以更全面性的感受法式彩妝精品的優質身段與多彩浪漫，接收MAKE UP FOR EVER最完整、最全方位的專業彩妝商品新知，接下來MAKE UP FOR EVER將一步步引領全球時尚、舞台、電影、彩妝藝術等風騷，挑動你對美的原始感知，和全球專業彩妝時尚徹底接軌。

全新MAKE UP FOR EVER團隊 與LVMH並肩揮灑亞洲彩妝時尚 
MAKE UP FOR EVER在台灣的10年，以專業通路起家，成為專業彩妝師最愛用品牌，用優秀專業的全方位彩妝商品與消費者的堅強口碑，每年以雙位數成長！多年來一路打造MAKE UP FOR EVER成為明星品牌的MAKE UP FOR EVER台灣區品牌總經理 陳佩琳信心喊話：「有國際時尚龍頭LVMH集團的相輔相成，全新專業團隊齊心經營，MAKE UP FOR EVER台灣勢必再繼續締造品牌神話！」絕對的色彩、豐富的彩妝語彙，法式奢華的剛勁與柔媚，透過專業頂尖團隊以及最堅強的時尚集團輝煌登場，讓我們拭目以待！
