來自真實故事的靈感
一切都是從MAKE UP FOR EVER創辦人Dany Sanz和巴黎最著名的歌舞表演團體MOULIN ROUGER紅磨坊芭蕾舞總監Janet Pharaoh的邂逅而開始。二位在各自藝術領域上，都擁有追求完美熱忱的女人，不僅彼此分享她們對舞台和表演藝術的熱愛，對於如何表現女人的美麗，也都有著高度敏銳的直覺。

就在這二位藝術家靈感激盪的合作過程中，Janet Pharaoh向Dany表達了希望能設計一款色彩飽合度更高，在舞台強光照射下，依然能完美顯色的唇膏，讓即使坐在遠處的觀眾，也能清楚看到舞者美麗妝容的產品，恰巧當時Dany正在研發MAKE UP FOR EVER即將於2010年10月上市的全新系列唇膏，不謀而合的火花，因此創造出4款今年秋冬最性感耀眼的新品－MAKE UP FOR EVER「紅磨坊限量版唇膏」、「紅磨坊限量版晶亮豐盈唇露」、「紅磨坊限量版派對型假睫毛」及「紅磨坊限量版水晶寶石」。
 
為了彰顯這個全新系列的獨特性，MAKE UP FOR EVER在MOULIN ROUGER紅磨坊限量系列的包裝上，也首次突破以往一貫全黑的外觀設計，不僅加入紅色的元素，更印上了MOULIN ROUGER紅磨坊的logo，以及MOULIN ROUGER紅磨坊最具代表性的風車圖騰，讓MAKE UP FOR EVER和擁有百年風華歷史地位的MOULIN ROUGER紅磨坊之間緊密連結的關係，從此確立無疑。

2010年秋冬時尚主打色：「ROUGE紅」
「紅色」，幾乎已是時尚伸展台上與黑色齊名的經典，在2010年秋冬，更將引爆大流行，從CHRISTIAN DIOR、GIVENCHY、PRADA到DONNA KARAN等各大精品時裝伸展台上，紅色元素幾乎無處不在，而MAKE UP FOR EVER得自紅磨坊舞者妝容靈感的MOULINE ROUGER紅磨坊限量系列，也將領著彩妝界的潮流，把「紅色」帶進更實用的生活層面，讓女人都能藉著自己的彩妝方法，詮釋出心裡最渴望的紅色風格。

許多人總有錯誤的想法，以為紅色容易流於俗氣或顯老，但其實只要依年紀和膚色，挑對適合的深淺和色調，紅色是最能帶出整體氣色的選擇，MAKE UP FOR EVER藝術暨創意總監洪祥瀚表示，膚色較白的人，正紅色的「紅磨坊限量版唇膏」妝效保證驚豔，而膚色偏黑的人，則可利用「紅磨坊限量版晶亮豐盈唇露」把唇色的亮度更提高，使整個人更顯明亮，不過，如果嘴唇較厚或嘴型較大者，唇露的用量拿捏則要特別留意。

和時尚紅唇最搭配的妝感，是乾淨的底妝與柔和的眼眸，帶有亮澤的古銅金、粉金或藕色，在眼窩上暈出淡淡的漸層，就能呈現出極度自然優雅的質感，想再華麗一些嗎？鑲有紅色亮片眼線的「紅磨坊限量版派對型假睫毛」，和採用Swarovski施華洛世奇最頂級水晶的「紅磨坊限量版水晶寶石」，立刻就讓你化身夜巴黎最亮眼的經典紅伶。

Beauty Story：MAKE UP FOR EVER與MOULINE ROUGE
早在2007年，MAKE UP FOR EVER和MOULINE ROUGE紅磨坊之間的合作就已展開，當時MAKE UP FOR EVER創辦人Dany Sanz為MOULINE ROUGE的舞者們設計了一款舞台表演專用的粉餅（PAN CAKE），絕佳的覆蓋度和完美符合舞台燈光需求的妝效，受到極大的好評，從此奠定雙方密不可分的合作關係。

至今，MOULINE ROUGE紅磨坊的舞者們，在成為能正式登台表演的舞者之前，除了要經過一連串嚴格的舞蹈表演訓練外，還必需到MAKE UP FOR EVER位在巴黎的BOUTIQUE接受專業的彩妝訓練，因為舞台妝和一般彩妝，在技巧上是是截然不同的，更大的眼睛、更濃的底妝和更重的修容，才能在舞台上呈現出明顯可見的妝感。現在MOULINE ROUGE紅磨坊正在上演的歌舞劇「Feerie」（仙境）彩妝造型正是由Dany所設計，而她目前也持續和Janet Pharaoh共同合作，為2010年MOULINE ROUGE紅磨坊即將新登場，演出時間長達十年的新作品設計彩妝造型。

MAKE UP FOR EVER的彩妝品不僅是MOULINE ROUGE紅磨坊舞者們唯一必備彩妝品外，在她們的日常彩妝中，MAKE UP FOR EVER同樣深受肯定，除了新上市的紅磨坊系列外，防水眼線筆、HD無瑕粉底液等，也都是她們的最愛。
