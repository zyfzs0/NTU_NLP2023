2012 年法國美妝領導品牌 LANCOME 推出第一款能將幸福與快樂儲存在瓶子裡的美好香水 - 美好人生( La vie est belle )，為所有女性寫下追求人生美好的美麗故事，美好人生 ( La vie est belle ) 是一種態度，更是一種新世紀的宣言，是充滿對人生中所有美好的人、事、物的追求，是一種幸福與自由的見證，然後美好人生 ( La vie est belle )。

一直以來，LANCOME仍持續傾聽女性內心的渴望，2014年美好人生 ( La vie est belle )的全新篇章，綻放屬於女性於陽光下開朗自然的並且展現自信的溫柔芬芳，再一次地將快樂與幸福做收藏，伴隨著清新溫暖，同時歡欣閃耀的自在美麗，全新的美好人生淡香氛，將再開啟女性一節全新的幸福篇章。


春日之花-木蘭
只因春日而甦醒、只因春日而綻放、只因春日而芬芳
根據植物科學記載，木蘭原生於遠東潮濕的樹木繁盛地區，許多國家文化都以其為尊貴與純潔的象徵。傳說中，十八世紀有位植物學家帶著木蘭的瓣狀被片，遠渡重洋來到歐洲沿海地區栽種在城堡花園中，卻因木蘭長期沒有再次開花因此很快就被遺忘。正當城堡主人打算丟掉它的同時，春天的第一道曙光灑下，木蘭的生氣和新芽因此而綻放，很快地，這當時看似已枯萎的木蘭，就隨著春日陽光的滋潤進而盛開、綻放。因此，木蘭身為因為春天陽光灑下後第一朵甦醒的花，故有「春日之花」的美名，高貴的木蘭用它盛開的的花瓣，回報春日的溫暖，木蘭更成為展現的同義詞，每一片木蘭花瓣紛紛隨著春日陽光的撒下伸展舞動，並且散發令人沉醉的清新香氛。


閃耀快樂與幸福的微笑瓶身
更深刻、更耀眼
春日下　因快樂而揚起的幸福微笑
乘著白色羽翼自由飛翔
靈感來自一個女性因快樂而揚起的幸福微笑，LANCOME將女性優雅的微笑嵌入玻璃中，這一款名為水晶微笑 ( Le sourire dec r i s t a l ) 瓶身設計來自LANCOME創辦人艾爾芒‧ 博蒂尚 (Armand Petitjean ) 於品牌初創之際的概念，終將於2013年推出的美好人生香水 ( Le sourire de cristal ) 得到實踐。延續2013年水晶微笑 ( Le sourire de cristal ) 的概念，2014 年全新的美好人生淡香氛的水晶微笑有了全新概念，瓶身採取縱向延伸，進而讓這經典的水晶微笑更閃耀、更陽光，宛如因為沐浴在溫暖春日下揚起的幸福微笑，更加的深刻、耀眼。原本珠光灰色的薄紗緞帶，也在2014年全新的美好人生淡香氛更換成純白色善要珠光的小絲巾，更加的符合這宛如沐浴在春日陽光下的幸福香氛以及溫暖感受，絲巾的尾端收尾宛如兩枚飄逸的薄翅構成春日下溫暖且幸福的心情飛揚剪影，讓人能夠乘著這白色雙翼自由翱翔，展開全新的美好人生 ( La vie est belle )！


春日下的自然、自在、自信
美好人生 ( La vie est belle ) 的幸福代表
茱麗亞．羅勃茲 ( Julia Roberts )
從經典的《麻雀變鳳凰Pretty Woman》到2001年的《永不妥協Erin Brockovich》為茱麗亞．羅勃茲 (Julia Roberts ) 拿一座奧斯卡金像獎最佳女主角，茱麗亞．羅勃茲的自然、自在與自信儼然就是美好人生( La vie est belle ) 的最佳幸福代表，特別是美好人生 ( La vie est belle ) 經典的水晶微笑(Le sourire de cristal )，完全性的呼應茱麗亞．羅勃茲的自然、自在與自信笑容。

2012年美好人生 ( La vie est belle ) 的推出，茱麗亞．羅勃茲的自然、自信以及自信無疑是最佳的代言人，茱麗亞．羅勃茲不僅在每一部電影中投入獨特並且真實的力量，延伸到戲外，更是自然、自信以及自信的絕佳代表，對於美好人生 ( La vie est belle ) 無疑是最完美的詮釋。2014 年全新的美好人生淡香氛，茱麗亞．羅勃茲再一次地用光彩奪目，並且舉手投足都是充滿自信自在的笑容為全新的美好人生淡香氛做完美詮釋。

如同迎接春日般的喜悅，並且擁抱春日來臨的第一道曙光，美好人生 ( La vie est belle ) 全新的幸福篇章-美好人生淡香氛，再一次由茱麗亞．羅勃茲為我們揭幕，宛如春日下的自然、自在和自信，是一種愉悅香氛，同時更是乘著花香微風而來的幸福感受，同時也在再一次的於全新幸福快樂宣言上，展現自然、自在和自信的微笑！
